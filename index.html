HTML

O que é HTML?

O HTML é uma linguagem de marcação que define a estrutura de elementos de uma página web. Pense no HTML como o esqueleto de uma página.

HTML é uma linguagem de programação?

Apesar de ter "linguagem" no nome, o HTML não é uma linguagem de programação. Isso porque no HTML não há tomada de decisões, não se processa dados, não utiliza variáveis e não tem condicionais! O HTML apenas marca a estrutura das páginas web.

Tags HTML

O que são tags?

Tags são elementos usados para marcar a estrutura do documento HTML.

Aqui está uma definição do site da Alura:

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros. As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Quais são as tags HTML?

Existem várias – várias mesmo! – tags HTML. Abaixo estão apenas algumas delas:

title: título do documento que aparece na aba do navegador
body: informações a serem vistas pelo usuário
main: principal conteúdo único da página
header: cabeçalho
footer: rodapé
section: define uma seção; pode estar dentro de outras seções
nav: navegação
strong: negrito
em: itálico
q: citação
abbr: abreviação
br: quebra de linha
